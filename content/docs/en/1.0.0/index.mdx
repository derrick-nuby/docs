# Application Overview

Performance Based Financing (PBF) rewards facilities for the services they deliver and the quality of care provided. The dashboard links data entry, quality scoring, administration tools, and reporting into a single workspace so teams can follow the full payment cycle.

## Navigating the Interface

- The home page presents quick links to common tasks.
- Breadcrumbs show where you are inside the application.
- A left side menu groups features by function and can be collapsed.
- The search bar helps locate facilities or records quickly.
- Icons highlight notifications and pending actions.
- Language options in the footer let users switch translations.
- Tooltips explain unfamiliar buttons and fields.

## Key Sections

- **Data Collection** stores service statistics submitted by facilities.
- **Quality Scores** list verification results and feedback.
- **Administration** holds configuration tools for the program.
- **Reports** summarize progress for managers and funders.
- **Help** links provide contact information and definitions.

## Tips for New Users

- Start with a guided tour to see how pages connect.
- Confirm your profile and facility assignment after logging in.
- Read tooltips carefully before entering numbers.
- Save drafts often to avoid losing work.
- Use export options to download summaries for local records.
- Check calendars for reporting deadlines and plan accordingly.
